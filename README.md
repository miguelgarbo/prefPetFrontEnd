# Projeto PrefPet Front End usando Angular 19

## O PrefPet é um site que foca na digitalização da carteira de vacinação do pet e na organização/centralização de serviços disponíveis para animais domésticos que existem na internet.

Tipos de Usuários: Tutor, Veterinário, Entidade, Admin

Tutor Funcionalidades = Cadastro de seus pets, Gerar a Carteira de Vacinação, Acompanhar as aplicações/vacinas que o pet já tomou ou precisá tomar, 
Visualizar Emergencias cadastradas com os respectivos contatos, Buscar qualquer Animal Por Microchip, Tranferir Tutela do pet, Recebimento de Notificações do sistema/convites de transferência de tutela,
Acompanhar Publicações das Entidades sobre as campanhas locais.

Veterinário Funcionalidades = Cadastro de Aplicação de Vacina de um animal do sistema, Visualização do Histórico das Aplicações

Entidade Funcionalidades = Cadastro de Publicações

Para inicializar o projeto

Step 1 - Baixar Dependencias 

run: npm install

Step 2 - Rodar Servidor de Desenvolvimento - Porta 4200

run: npm s -o
